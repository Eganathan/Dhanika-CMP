# Dhanika

The project intends to help us to live on a budget, with consciously helping us create/track and analyse our budgets and eventually help us to get some clarity on our financial goals.

This is a Kotlin Multiplatform project targeting Android, iOS, Desktop (JVM), and is my way of learning this technology, you are most welcome to contribute, by raising issues and sharing feedbacks to improve this codebase so more people can use it effectively.
